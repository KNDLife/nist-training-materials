# NIST Training Materials (2024-2025 Updates)

This repository contains concise, one-page summaries of the latest NIST standards
and guidance updates from 2024-2025.  
Each document is written in plain English for a general audience.

## Contents
- SP 800-53 Release 5.2.0 – Software Update & Patch Management  
- SP 800-232 – Lightweight Cryptography (Ascon)  
- SP 800-228 – API Protection for Cloud-Native Systems  
- SP 800-171r3 & 172r3 – Controlled Unclassified Information (CUI)  
- SP 800-63-4 – Digital Identity Guidelines  
- SP 800-88r2 – Media Sanitization  
- SP 800-61r3 – Incident Response  
- SP 800-55 – Information Security Measurement  
- SP 800-234 – HPC Security Overlay  
- SP 800-18r2 – Security/Privacy/SCRM Plan  
- SP 800-189r1 – BGP Security  
- SP 800-90C – Random Bit Generators  
- AI System Security Overlays  
- Post-Quantum Cryptography (FIPS 203-205, SP 800-227)  
- Future Drafts (Rijndael-256 & SP 1331 CSF 2.0 Quick-Start)

## Usage
You may copy, distribute, and adapt these materials for educational or
organizational training.  Attribution to NIST is appreciated where appropriate.
